BigDecimal 舍入模式

ROUND_UP：舍入远离 0

ROUND_DOWN：舍入靠近 0

ROUND_CEILING：向上（正数方向）舍入

ROUND_FLOOR：向下（负数方向）舍入

ROUND_HALF_UP：四舍五入

ROUND_HALF_DOWN：五舍六入

ROUND_HALF_EVEN：奇（四舍五入）偶（五舍六入）舍入

ROUND_UNNECESSARY：非必要舍入（固定格式）
